## üéØ Usage Guide

### How to Generate (Step-by-Step)

1. **Go to Gemini Chat**
   - Visit: https://gemini.google.com/
   - Sign in with Google account

2. **Enable Imagen 3**
   - Ensure you're using Gemini Pro with Imagen support
   - (Should be enabled by default in free tier)

3. **Copy & Paste Prompt**
   - Choose a prompt from above
   - Paste directly into Gemini chat
   - Send message

4. **Review & Iterate**
   - Gemini will generate 1-2 images
   - If not satisfied, refine prompt:
     - "Make it more colorful"
     - "Simplify the design"
     - "Change the color to [hex code]"

5. **Download Images**
   - Right-click ‚Üí Save image
   - Or click download button

6. **Optimize for Web**
   - Use TinyPNG.com or ImageOptim to compress
   - Target: <300KB per image

7. **Add to Project**
   ```bash
   # Create assets folder
   mkdir -p assets/branding
   
   # Add images
   cp ~/Downloads/elimu_icon.png assets/branding/icon.png
   cp ~/Downloads/elimu_banner.png assets/branding/banner.png
   ```

## üé® Design Tips

### Color Accessibility
- Ensure contrast ratio ‚â• 4.5:1 for text
- Test with color blindness simulators
- Purple/pink should work for protanopia/deuteranopia

### Icon Best Practices
- Keep it simple (max 3 elements)
- Avoid thin lines (<2px at small sizes)
- Test at 16x16, 48x48, and 1024x1024
- Ensure it works in light AND dark mode

### File Formats
- **Icons:** PNG with transparency
- **Banners:** JPG (smaller file size) or PNG
- **Logos:** SVG (scalable) + PNG fallback
